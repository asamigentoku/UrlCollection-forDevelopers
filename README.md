# UrlCollection-forDevelopers

AI時代だからこそ必要となる、開発者のための「公式ドキュメント」特化型Webリンク集です。生成AIによるハルシネーション（誤情報）に惑わされず、常に一次情報（正しい知識）へ素早くアクセスできる環境を提供します。

海外のエンジニアもスムーズに利用できるよう、言語設定が可能です

---

## 🚀 主な機能 (Features)

* **公式ドキュメント限定:** 信頼できる一次情報のみを厳選して掲載。
* **データの自動更新:** バックエンドで定期的にデータが同期され、常に最新の状態を維持。
* **グローバル対応:** 海外エンジニアの利用を想定した、英語ファーストのインターフェース。

---

## 🛠️ アーキテクチャ & 技術スタック (Tech Stack)

パフォーマンス、低コスト、運用の手軽さを両立するため、モダンなサーバーレスアーキテクチャを採用しています。

### フロントエンド
* **React:** 開発者が扱いやすい、クリーンでレスポンシブなUI。
* **多言語対応:** グローバル対応を見据えた英語表記。

### バックエンド & データパイプライン
* **AWS Lambda (Python):** 定期実行（Cronジョブ）により、スプレッドシートのデータを自動で読み込み、Firebaseへ書き込むサーバーレス関数。
* **Google Sheets API:** 管理者側でURLを簡単に追加・編集できるよう、軽量なCMS（コンテンツ管理システム）としてGoogleスプレッドシートを活用。
* **Firebase (Firestore):** フロントエンドへ高速にデータを配信するためのメインデータベース。

---

## 🔄 データフロー (Data Flow)

1. **データ管理:** 管理者が **Googleスプレッドシート** に公式ドキュメントのURLや情報を入力・更新します。
2. **定期実行:** **AWS Lambda**（Python）が定期的に起動します。
3. **同期:** LambdaがGoogle Sheets API経由でデータを取得し、**Firebase (Firestore)** へデータを上書き・更新します。
4. **配信:** **React**で作られたフロントエンドが、Firebaseから最新のデータを直接取得して画面に表示します。
